Example: Top-in-at-Random

考虑一种洗牌方式：每次把最顶上的牌插入随机位置。

Claim: 这种洗牌方式的混合时间为 O(nlog⁡n)O(n \log n)O(nlogn)。

用 TTT 表示原本最底下的牌被随机插入的时刻，则 TTT 是一个强稳定时间。可见 T=T1+T2+⋯+Tn−1+1T = T_1 + T_2 + \cdots + T_{n-1} + 1T=T1​+T2​+⋯+Tn−1​+1，其中 TiT_iTi​ 表示从位置 iii 变动到 i+1i+1i+1 所需要的时间。每个 TiT_iTi​ 的分布是几何分布，期望为 n/in/in/i，故 E[T]=O(nlog⁡n)\mathbb E[T] = O(n \log n)E[T]=O(nlogn)。根据 Markov 不等式，τmix≤O(nlog⁡n)\tau_{\rm mix} \le O(n \log n)τmix​≤O(nlogn)。

Example: Random Transposition Shuffle
