PolarNet

本文提出了 PolarNet，一种基于点云的语言引导机器人操作网络，基于 PointNext 架构 [17]，以高效编码点云输入。通过多模态 Transformer，在中间层将编码后的点云与语言指令融合，预测 7-DoF 的 action。

目标是学习一个视觉策略 π(at∣Ot)\pi(a_t | O_t)π(at​∣Ot​)，其中 ata_tat​ 是动作，OtO_tOt​ 是 ttt 时间的观察空间，包括：

  1. 语言指令 (x1,⋯ ,xNx)(x_1, \cdots, x_{N_x})(x1​,⋯,xNx​​)，xix_ixi​ 是 token 化的单词
  2. KKK 个摄像头的 RGB 图像 {Irgbk∈RH×W×3}k=1K\{I_{rgb}^{k} \in \R^{H\times W \times 3}\}_{k=1}^{K}{Irgbk​∈RH×W×3}k=1K​
  3. 对齐的深度图像 {Idepk∈RH×W}k=1K\{I_{dep}^{k} \in \R^{H \times W}\}_{k=1}^{K}{Idepk​∈RH×W}k=1K​

Point Cloud Inputs

将深度图像中的每个像素投影到 3D 世界坐标系中，每个像素的 RGB 颜色可以附加到对应的 3D 点上，为每个摄像头生成一个点云 Vtk∈RH×W×6V_{t}^{k} \in \R^{H\times W \times 6}Vtk​∈RH×W×6，每个点由 XYZ 坐标和 RGB 颜色组成。

将它们合并为一个统一的点云。为了减少冗余，使用 Open3D [52] 对合并后的点云进行均匀下采样，每个体素保留一个点。同时定义了一个覆盖桌面上方工作空间的 3D 边界框，用于裁剪点云，仅保留物体和机器人手臂的点。最终得到的点云 VtV_tVt​

实验证明，下面 4 种输入特征均有助于性能提升，其中颜色尤为重要：

  1. XYZ 坐标
  2. RGB 颜色
  3. 法向量：已被证明在避障任务中有效 [45]。
  4. 相对于操作桌面的高度

Model Architecture

采用 CLIP 模型 [8] 的语言编码器对语言指令进行标记化和编码。冻结 CLIP 的参数，并添加一个线性层以生成语言嵌入。

X^=(x^1,⋯ ,x^Nx)=WxCLIP([x1,⋯ ,xNx])\hat{X} = (\hat{x}_1, \cdots, \hat{x}_{N_x}) = W_x \text{CLIP}([x_1, \cdots, x_{N_x}])X^=(x^1​,⋯,x^Nx​​)=Wx​CLIP([x1​,⋯,xNx​​])

使用 PointNext 编码器 [17] 对点云 VtV_tVt​ 进行编码，PointNext 编码器由 LeL_eLe​ 个 Set Abstraction (SA) 模块组成，用于分层抽象点云特征。每个 SA 模块包含以下步骤

  1. 子采样层：用 FPS 采样 NvlN_v^lNvl​ 个点
  2. 分组层：为每个采样点找到邻居点，即距离 rlr^lrl 之内的点
  3. MLP
  4. Max Pooling

假设查询点 iii 在第 lll 层的坐标为 pilp_i^lpil​，输入特征 vilv_i^lvil​，则

每个 SA 模块将点的数量减少一半，同时将特征维度增加一倍。我们将编码后的点云表示为 VtLeV_t^{L_e}VtLe​​。

用多层 Transformer 架构 [53] 将点云特征与语言嵌入融合。首先为点云特征添加位置编码和正弦位置嵌入。
